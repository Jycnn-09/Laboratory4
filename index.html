User Guide

  • Click number buttons (0–9) to input values.
  • Select an operator (+, −, ×, ÷) to perform calculations.
  • Press "=" to evaluate the expression.
  • Click "C" to clear all input.
  • Click "⌫" to delete the last character entered.

Keyboard Controls

  • Use number keys (0–9) for input.
  • Use +, -, *, / for operations.
  • Press Enter to calculate.
  • Press Backspace to delete.
  • Press C to clear.
